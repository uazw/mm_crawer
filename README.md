mm_crawer
=========

##mm_crawer
本项目是基于scrapy,进行简单的封装
在mm_crawer下
  python main.py
##思路
1.分析网站结构 得到需要提取数据的页面（用浏览器）的跳转路径  
2.结合xpath分析出各个跳转点需获取的url或者数据  
3.实现基本脚本  
以上为无js渲染的情况下  

4.关闭浏览器对JS渲染的支持 得到结论(例如在本网站中图片链接是用过js渲染得到)  
5.模拟JS渲染效果(这个网站把图片链接写在script内 用正则匹配即可)
也有部分网站是异步获取(这时用浏览器的netowrk功能分析异步的链接)
想办法拼接链接即可
6.实现其他功能(例如在本例 命令行帮助)
